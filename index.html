Grammar & Particle Bridge (MM ⇄ JP)

Hacking Japanese vocabulary: identical word order (SOV) and particle-for-particle translation.

1. Particle Hack Shortcut

Japanese Particle Myanmar Equivalent Function / Usage Example
は (wa) / が (ga) သည် (thi) / က (ka) Subject / Topic marker わたしは (ကျွန်တော်သည်)
を (wo) ကို (ko) Direct object marker ごはんを (ထမင်းကို)
に (ni) / へ (e) သို့ (tho) / ဆီသို့ Destination / Time / Location がっこうへ (ကျောင်းသို့)
で (de) မှာ (hmar) / နဲ့ (nae) Action location / Means / Tool うちで (အိမ်မှာ) / バスで (ဘတ်စ်ကားနဲ့)
の (no) ၏ (ei) / ရဲ့ (yae) Possession (of / 's) わたしのねこ (ကျွန်တော်ရဲ့ကြောင်)
から (kara) ... まで (made) မှ (hma) ... အထိ (a-hti) From ... To (Range) 9時から (၉နာရီမှ) 5時まで (၅နာရီအထိ)

2. 1:1 Word Order Sentence Match

Since both languages are SOV (Subject-Object-Verb), you can construct Japanese sentences by translating word-by-word from Myanmar!

Example 1: "I eat rice."
わたしはကျွန်တော်သည်I (subject)
ごはんをထမင်းကိုrice (object)
たべます。စားပါသည်၊၊eat (verb)
Example 2: "I go to school by bus."
わたしはကျွန်တော်သည်I (subject)
バスでဘတ်စ်ကားဖြင့်by bus (means)
がっこうへကျောင်းသို့to school (dest)
いきます。သွားပါသည်၊၊go (verb)

3. Japanese Verb Types & Conjugation Hack

Japanese verbs have 3 main groups. Suffix conjugations map 1:1 to Myanmar verb suffixes!

Japanese Verbs (動詞 - Doushi)
Group 1 (Godan / -u)
Ends in ~u (e.g. 飲む Nomu)
Polite: 飲みます (-imasu)
Negative: 飲みません (-imasen)
Past: 飲みました (-imashita)
Group 2 (Ichidan / -ru)
Ends in ~iru / ~eru (e.g. 食べる Taberu)
Polite: 食べます (-masu)
Negative: 食べません (-masen)
Past: 食べました (-mashita)
Group 3 (Irregular)
する (To Do) / くる (To Come)
Polite: します / きます
Negative: しません / きません
Past: しました / きました
飲む (のむ - Nomu) ⇄ သောက်သည် (Drink)
